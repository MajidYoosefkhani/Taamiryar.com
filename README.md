# taamiryar.com 🔧

دستیار هوشمند تعمیرات ایران

## راه‌اندازی روی Vercel

### ۱. نصب وابستگی‌ها

```bash
npm install
```

### ۲. تست محلی

```bash
npm run dev
```

### ۳. دیپلوی روی Vercel

```bash
npm install -g vercel
vercel
```

### ۴. وصل کردن دامنه

- در داشبورد Vercel برو به Settings > Domains
- آدرس `taamiryar.com` رو اضافه کن
- در Namecheap DNS رو تنظیم کن

### ۵. درگاه تیپ

در فایل `src/App.jsx` آدرس Buy Me a Coffee رو جایگزین کن:

```
https://buymeacoffee.com/taamiryar
```
